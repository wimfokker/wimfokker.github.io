Mijn projecten.

Dit zijn de projecten wat ik de eerste periode heeft aan gewerkt op school. Op basisprogrameren moest ik het spel MasterMind maken in Java. Voor Website heb ik trainings site gebruik van Flexbox Defence, Flexbox froggy om meer over flex te weten, en voor een opdracht heb ik Flex blokken gemaaakt om het aan zetten van blokken in een container. Voor website heb ik ook een website gemaakt om een design die PinkGoose heet. Voor project week had ik met een groep gewerkt om een ROC website te bouwen in HTML en CSS wat een best wel goed ging.

MasterMind

In deze project heb ik gemaakt op Java, het begon eerst van het maken met Pseudocode of wel in menselijketaal in een Words bestand van hoe de spel Mastermind werkt en van daar ging de code die we hebben gemaakt naar een Flowgorithm brengen en met het stappen plannen klaar ging we Mastermind maken op Eclipse programma van Java.

Flexbox Defense

Flexbox Defense was een opdracht voor de eerste periode van website in de begin. Het is een leerzaam spel om dat het elementen van een tower defense game combineerd met het leren van CSS code en dat had ik afgemaakt.

Flexbox Froggy

Flexbox froggy was ook net als Flexbox defense een leerzaam spel als een interactieve manier om css flexbox te leren met 24 levels. In de begin vond ik CSS moelijk maar dit had wel geholpen om meer te weten over CSS.

PinkGoose website

De PinkGoose website was een project om een website bestand op Figma na te maken op Visual studio code met HTML en CSS. In deze project ging ik meer beginnen van het maken op stappen en containers van een pagina na te maken in HTML en CSS.

Project week

In project week heb ik met een groep gewerkt in twee andere studenten in mijn klas. Wat wij moesten maken was een ROC website in de projectweek, wij begonnen eerst met een Trello te maken en daar de ideeën zetten voor de website en van daar gingen wij met een team taken verdelen met wie doet de CSS en wie gaat informatie verzalamen.

Flex blokken

Flex blokken was een opdracht met HTML en CSS van het zetten van blokken in een container voor de eerste periode van website. in deze project had ik een HTML en CSS gezet en het kijken in een work folder welke positie de blokken moesten zijn en dat daarna na gemaakt.
